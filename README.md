Требования для установки
------------

  -PHP 7;
  
  -Composer;

Запуск
---------------
Скачать проект, распокавать архив, выполнить 

```
cd GildedRose-Refactoring-Kata-php-master
composer install
vendor/bin/phpunit
php fixtures/texttest_fixture.php N где N - количество дней для вывода отчёта
```
